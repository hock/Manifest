Manifest Documents

Manifest has been deployed in a variety of different contexts, from academic research projects and courses to workshops for organizers and activists. This page collects only those Manifests and datasets that have been identified for broad publication.

One note about this collection is that Manifest is not a database. User-created documents are not shared with the Manifest install at manifest.supplystudies.com and the documents on this page include only those that have been explicitly shared for publication. If you have created Manifest documents or have deployed Manifest in classrooms or workshops and would like the results indexed here, please get in touch.

      Sourcemap Archive

      This is an archive of the data collected by the MIT Center for Civic Media's Sourcmap Project from 2006-2014. That means that this data is old. Because Sourcemap evolved significantly during this relatively long span of time, and because one of the development goals was to avoid breaking user projects, it also means this data is inconsistent. Some attributes and properties that were added in, for example, 2008, probably didn't exist in 2006. And it is also true that some early attributes and properties were depreciated or changed over time. While these have been preserved in the raw data files referenced below, only a subset of more or less standard information is directly viewable in Manfest.
