Manifest JSON

This is a simple editor for creating Manifest supply chains. These are just JSON files that correspond to the Manifest Schema. This is the preferred mechanism for modeling supply chains, but the format is still in development. it isn't the only option. Manifest also supports the SMAP geojson format that was originally created for the Sourcemap project.

Google Sheets

An alternative to using Manifest JSON is to create your supply chain in a Google Sheet. We provide a simple Google Spreadsheet template that can be loaded into Manifest by making the sheet public and passing its ID to Manifest.
