This is a simple editor for creating Manifest supply chains. These are just JSON files that correspond to the Manifest Schema. This is the preferred mechanism for modeling supply chains going forward, but this format is still in development. it isn't the only option. Manifest also supports data that was originally created for the Sourcemap project, and a simple Google Spreadsheet template.
